# Design References

This folder contains design reference images for UI redesign work.

## Required Images

Please save the following design mockup images to this folder:

### 1. Dashboard Redesign (`dashboard-redesign.png`)
- **Issue**: #139
- **Description**: Clean dashboard design with:
  - Welcome message header
  - Quick start card with CTA
  - Topic cards in grid layout (Spanish, German)
  - Rounded corners and modern card design
  - Bottom navigation bar

### 2. Practice Session Redesign (`practice-session-redesign.png`)
- **Issue**: #140
- **Description**: Mobile-first practice session design with:
  - Clean question header
  - Multiple choice options as full-width buttons
  - Action buttons at bottom (Check, Hint, Skip)
  - Simple navigation back button

## How to Add Images

1. Save the design mockup images to this folder
2. Use the filenames specified above
3. Update the GitHub issues #139 and #140 with the image paths:
   ```markdown
   ![Dashboard Design](docs/design-references/dashboard-redesign.png)
   ![Practice Session Design](docs/design-references/practice-session-redesign.png)
   ```
